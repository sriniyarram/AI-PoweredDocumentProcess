  Solution: Kill the process using that port (see Common Tasks above)
            Or use a different port by modifying the server code

  Solution: Install from https://www.python.org/ or https://nodejs.org/
            Make sure to add to PATH during installation

Problem: "Cannot connect to server"
  Solution: 1. Verify server is running (check terminal)
           2. Check correct URL (http://localhost:3000 or 8080)
           3. Refresh browser (Ctrl+R or Cmd+R)
           4. Check firewall settings

  Solution: Both servers have CORS enabled - should work fine
            Try refreshing browser and clearing cache

Problem: "Documents disappear after restart"
  Solution: This is expected! Data is stored in memory only
           Implement database to persist data (see SETUP.md)
